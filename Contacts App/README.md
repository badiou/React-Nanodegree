# Contacts

This is a code-along with the in-course project in the React Nanodegree program.

## Project Setup

1. Go to the 'Starter' or 'Finished' folder

2. Install the dependencies.

```bash
npm install
```

3. Start the development server.

```bash
npm run start
```

4. Open your browser and navigate to the development server URL (e.g., `http://localhost:3001`).
